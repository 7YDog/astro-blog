# <https://yutengjing.com>

基于 [astro](https://github.com/withastro/astro) 和 [astro paper](https://github.com/satnaing/astro-paper) 模版，使用 [cloudflare pages](https://dash.cloudflare.com/) 部署。

## 本地开发

### 安装依赖

```bash
pnpm install
```

### 写作和预览

```bash
pnpm dev
```

### 生产构建

```bash
pnpm build
```
